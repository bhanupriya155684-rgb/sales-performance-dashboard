# Sales Performance Dashboard | Power BI & Excel
## Project Overview
Built an interactive Sales Performance Dashboard using Power BI 
on the Superstore dataset to analyze regional sales trends, 
product performance, and profitability.

## Tools Used
- Power BI Desktop
- Excel (Data Cleaning)
- DAX (Data Analysis Expressions)
- GitHub (Version Control)
## Dataset
- Source: Kaggle — Superstore Sales Dataset
- Records: 9,994 orders
- Period: 2014-2017

## Dashboard Pages
1. **Sales Overview** — KPI cards, sales trend, regional map
2. **Product Analysis** — Top 10 products, profit by sub-category
3. **Regional Analysis** — Sales vs profit, return rate by region
4. **YoY Growth Analysis** — Year over Year sales growth trends

## DAX Measures Created
- Total Sales, Total Profit, Total Orders
- Profit Margin %, Average Order Value
- YoY Sales Growth
- Return Rate %
- High Value Orders
- Discount Impact
- Lost Revenue Due to Discount

## Key Findings
- West region had highest sales
- Technology category most profitable
- Discount heavily impacts profit margin
- Sales grew consistently from 2015 to 2017
